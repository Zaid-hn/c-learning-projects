# ATM Simulator

A simple console-based ATM simulator written in C.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Prevent negative deposits
- Prevent withdrawing more than the available balance
- Menu-driven interface
- Continuous interaction until the user exits

## Concepts Practiced

- Variables
- User Input (`scanf`)
- Conditionals (`if` / `else`)
- Loops (`while`)
- Functions
- Basic Input Validation

## Future Improvements

- Save balance to a file
- PIN authentication
- Multiple bank accounts
- Transaction history
