# C Learning Projects

A collection of console-based C projects built while learning the C programming language.

The goal of this repository is to practice programming fundamentals, improve problem-solving skills, and document my learning journey through progressively more challenging projects.

## Projects

### ATM Simulator

A simple ATM system that allows users to:

- Check account balance
- Deposit money
- Withdraw money
- Validate user input

**Concepts practiced**

- Variables
- Conditionals
- Loops
- Functions

---

### Trivia Quiz

A console-based multiple-choice quiz game.

**Concepts practiced**

- User Input
- Conditionals
- Program Flow

---

### RPG Character Creator

A small RPG-style character creation program.

Features include:

- Character name input
- Three playable classes
- Difficulty selection
- Character sheet generation

**Concepts practiced**

- Structures
- Strings
- Arrays
- Two-dimensional Arrays
- Functions
- Pointers
- Passing structures to functions
- Data-driven programming
