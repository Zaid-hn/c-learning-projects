# Trivia Quiz

A simple console-based trivia quiz game written in C.

## Features

- Multiple-choice questions
- User answer checking
- Displays whether the answer is correct
- Simple terminal interface

## Concepts Practiced

- Variables
- Strings
- User Input
- Conditionals
- Program Flow

## Future Improvements

- Multiple questions
- Score tracking
- Randomized question order
- Difficulty levels

## Notes

This project was created while learning the C programming language. The focus was on understanding core programming concepts rather than building a production-ready application.

As I continue learning C, I plan to revisit and improve this project using more advanced techniques.
