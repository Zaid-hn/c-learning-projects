# RPG Character Creator

A console-based RPG character creator written in C.

The program allows the user to create a character by selecting a class and difficulty, then generates a character sheet with the appropriate stats.

## Features

- Enter player name
- Choose a character class
  - Warrior
  - Archer
  - Mage
- Choose difficulty
  - Easy
  - Medium
  - Hard
- Automatically assigns stats
- Displays a formatted character sheet

## Concepts Practiced

- Structures (`struct`)
- Functions
- Arrays
- Two-dimensional Arrays
- Strings
- Pointers
- Passing structures to functions
- Data-driven programming

## Project Structure

- Character information stored using a `struct`
- Character classes stored in arrays
- Character stats stored in a 2D array
- Functions used to separate program logic
